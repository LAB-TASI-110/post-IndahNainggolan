Prompt T02 (versi mahasiswa, tapi jelas & tajam) Aku lagi ngerjain T02, disuruh buat program Java untuk ngolah data stok barang di gudang. Di soal dikasih konsep rak penyimpanan, di mana tiap barang punya jumlah stok dan kategori tertentu. Tujuan program: Program harus bisa menghitung total stok hanya untuk kategori barang yang diminta user. Detail yang harus dikerjakan: Input: Jumlah data barang (N) Deret stok barang (s1, s2, ..., sN) Deret kode kategori tiap barang (panjangnya sama dengan N) 1 kode kategori yang ingin dicari total stoknya Contoh sederhana: N = 4 stok = 22 85 5 120 kategori = A B A C cari = A Berarti: Barang ke-1 kategori A → 22 Barang ke-3 kategori A → 5 Total = 27 Output: Total stok dari kategori yang diminta Konteks pengerjaan (PENTING): Bahasa: Java Struktur project sudah ada: driver/Driver2.java model/ (kalau perlu class tambahan) Jadi: Driver2.java = main program Kalau perlu, buat class tambahan di model Ketentuan implementasi: Gunakan array untuk menyimpan data Input pakai Scanner Gunakan perulangan (for) untuk proses Gunakan if / kondisi untuk cek kategori Jangan hardcode data Program harus fleksibel (sesuai input user) Yang aku butuhkan dari kamu: Kode lengkap Java (Driver2.java) Kalau perlu, pisahkan ke class di folder model Penjelasan singkat alur: input → proses → output Kasih contoh input & output biar aku bisa test
